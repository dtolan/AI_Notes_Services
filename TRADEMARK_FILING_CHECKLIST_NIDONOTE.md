## Application Process (USPTO)

### âœ… Step 4: File Application

**Filing Methods:**

**Option 1: TEAS Plus (Recommended - $350/class)**
- Lowest cost option
- Requirements:
  - Select descriptions from USPTO's Acceptable ID Manual
  - Agree to electronic correspondence
  - Pay entire fee upfront
  - Provide email for all communication
- **Total cost:** $700 for 2 classes (9 + 42)

**Option 2: TEAS Standard ($650/class)**
- More flexibility in descriptions
- Can use custom descriptions not in ID Manual
- **Total cost:** $1,300 for 2 classes
- **When to use:** If your description doesn't fit pre-approved options

**Filing Steps:**

1. **Access TEAS System**
   - URL: https://www.uspto.gov/trademarks/apply
   - Choose "TEAS Plus" application
   - Create USPTO.gov account

2. **Wizard Steps**
   - [ ] Step 1: Enter mark information
   - [ ] Step 2: Select applicant type (individual/company)
   - [ ] Step 3: Enter owner information
   - [ ] Step 4: Select goods/services classes
   - [ ] Step 5: Choose basis (1(a) or 1(b))
   - [ ] Step 6: Upload specimen (if 1(a))
   - [ ] Step 7: Review and validate

3. **Payment**
   - Credit card, debit card, or automated deposit account
   - Fee is non-refundable even if rejected

4. **Confirmation**

   - Save filing receipt PDF

**After Filing:**

### âœ… Step 5: Respond to Office Actions

**What is an Office Action?**
- Official letter from USPTO examining attorney
- May approve, request changes, or reject application
- **Deadline to respond:** 6 months (strict deadline, no extensions)

**Common Objections:**

1. **Descriptiveness**
   - Issue: "NIDONOTE" describes note-taking function
   - Response: Argue "NIDO" (nest) is arbitrary/fanciful for software
   - Likelihood: Low risk (NIDO is not descriptive in English)

2. **Likelihood of Confusion**
   - Issue: Similar to existing mark
   - Response: Argue differences in sound, appearance, meaning, channels of trade
   - Likelihood: Depends on search results

3. **Specimen Issues**
   - Issue: Specimen doesn't show mark in use
   - Response: Submit corrected specimen
   - Likelihood: Medium risk if using mock-ups

4. **Identification Issues**
   - Issue: Goods/services description unclear
   - Response: Clarify or narrow description
   - Likelihood: Low with TEAS Plus (pre-approved descriptions)

**Response Options:**
- [ ] DIY response (if simple clarification)
- [ ] Hire attorney for $500-1,500 per Office Action
- [ ] Abandon application (forfeit filing fee)

**Success Rate:**
- ~50% of applications receive at least one Office Action
- ~75% of applicants successfully overcome objections
- ~60% overall registration rate (first-time filers)

### âœ… Step 6: Publication & Opposition

**Publication for Opposition:**
- If approved, mark is published in **Official Gazette**
- **Opposition period:** 30 days for third parties to oppose
- Anyone can file opposition if they believe mark harms them

**Potential Opposition:**
- Existing "NIDO" trademark owners in other classes
- Note-taking app competitors claiming confusion
- Likelihood: Low (if search showed no conflicts)

**If Opposed:**
- Opposition is like a mini-trial before USPTO's Trademark Trial and Appeal Board (TTAB)
- **Cost:** $5,000-$25,000 in attorney fees
- **Duration:** 6-18 months
- **Outcome:** Settlement, cancellation, or registration

**If No Opposition:**
- Proceed to registration (1(a) filing) or Notice of Allowance (1(b) filing)

**If filed Use in Commerce (1(a)):**

### Filing Methods

**Option 1: Direct National Filing**
- File separately in each country
- **Pros:** Full control, tailored to each country
- **Cons:** Expensive, complex, different timelines
- **Cost:** $1,000-3,000 per country (including attorney fees)

**Option 2: Madrid Protocol (International Registration)**
- Single application covers 130+ countries
- Based on U.S. application (must file U.S. first)
- Managed through USPTO to WIPO (World Intellectual Property Organization)
- **Pros:** Cost-effective, centralized management
- **Cons:** Dependent on base application for first 5 years
- **Cost:** $650 base fee + ~$200 per country

**Recommended Strategy:**
1. **Now:** File in U.S. only
2. **6 months:** File in Canada (if app gains traction)
3. **12 months:** File Madrid Protocol for EU, UK, Australia (if revenue > $100k/year)

## Post-Filing Maintenance

### âœ… Ongoing Requirements

**Maintenance Deadlines:**

| Deadline | Requirement | Cost | Consequence if Missed |
|----------|-------------|------|----------------------|
| **5-6 years** after registration | Section 8 Declaration (proof of continued use) + Section 15 Declaration (incontestability) | $525 per class | Cancellation |
| **9-10 years** after registration | Renewal (Section 9) | $525 per class | Cancellation |

**Section 8 Declaration (Years 5-6):**
- File between 5th and 6th anniversary of registration
- Submit specimen showing continued use
- Declare mark is still in use in commerce
- **Grace period:** 6 months late filing allowed with $100 surcharge

**Section 15 Declaration (Years 5-6):**
- Optional but highly recommended
- File with Section 8 declaration
- Makes registration "incontestable" (harder to challenge)
- Strengthens legal rights

**Section 9 Renewal (Years 9-10):**
- File between 9th and 10th anniversary, then every 10 years
- Submit specimen showing continued use
- Can file combined with Section 8 if timing aligns

## Frequently Asked Questions

**Q: When should I file for trademark protection?**
A: File as soon as you've decided on the name and completed a search. You can file "Intent to Use" (1(b)) before launch to secure your priority date.

**Q: Can I file a trademark myself, or do I need a lawyer?**
A: You can file yourself (DIY) if you're confident in your search and the application is straightforward. However, an attorney increases success rate and can handle objections more effectively. Recommended if budget allows ($2,500-4,500).

**Q: What if someone else is using "NIDOnote" but hasn't filed a trademark?**
A: Common law trademark rights exist based on actual use. If they used it first in commerce, they may have superior rights even without registration. Consult an attorney to assess risk.

**Q: How long does trademark registration take?**
A: 12-18 months for "Use in Commerce" (1(a)) filings, 18-36 months for "Intent to Use" (1(b)) filings, assuming no significant objections or opposition.

**Q: What if my application is rejected?**
A: You can respond to Office Actions with arguments or amendments (deadline: 6 months). If finally rejected, you can appeal to TTAB or file a new application with changes. Filing fee is non-refundable.

**Q: Do I need to trademark "NIDOnote" in every country?**
A: No. Start with the U.S. (your primary market). File internationally (Canada, EU, etc.) once your app gains traction and revenue justifies the cost (~$1,000-2,000 per country).
